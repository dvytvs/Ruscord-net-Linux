Ruscord Net


Ruscord Net Linux — это неофициальный Linux-клиент Ruscoed Net для общения, вдохновлённый идеей Discord, но адаптированный под удобство и особенности Linux-систем.

🚀 О проекте

Ruscord Net создан как альтернатива, которой официально не существует для Linux. Приложение разработано на Electron и распространяется во всех популярных форматах, чтобы вы могли установить его на любой дистрибутив без лишних сложностей.

📦 Установка
Snap Store

Установить можно напрямую через Snap:
👉 https://snapcraft.io/ruscord-net

sudo snap install ruscord-net
